# 🧠 موسوعة الـ 300 مغالطة منطقية

أكبر مرجع عربي تفاعلي يجمع أشهر الأخطاء والتحيزات المنطقية التي نستخدمها في حياتنا اليومية وفي النقاشات العامة. تم إعداد هذا المشروع ليكون أداة بحثية لتعزيز التفكير النقدي وتفنيد الحجج الواهية.

## 🚀 رابط الموقع
[https://mabed4772-cloud.github.io/-/] (ضع رابط الموقع الذي أعطاك إياه جيت هاب هنا)

## ✨ المميزات
* **الشمولية:** يحتوي على 300 مغالطة منطقية كاملة مع شرحها.
* **أمثلة توضيحية:** مثالين لكل مغالطة لتبسيط الفكرة وربطها بالواقع.
* **البحث الفوري:** ميزة البحث الذكي التي تسمح لك بالوصول لأي مغالطة في أجزاء من الثانية.
* **تصميم مريح:** واجهة مستخدم بسيطة تدعم الوضع الليلي للقراءة الطويلة.

## 🛠️ كيف تستخدم هذا المشروع؟
هذا المشروع عبارة عن صفحة واحدة فقط `index.html`. يمكنك:
1. استخدامه كمرجع أثناء مشاهدة المناظرات أو قراءة المقالات.
2. البحث عن مغالطة معينة عندما تشعر بوجود فجوة في منطق شخص ما.
3. مشاركة الروابط مع الآخرين لرفع مستوى الوعي الفكري.

## 📝 ملاحظات للدارسين
تجد في نهاية الصفحة ملاحظات هامة حول تداخل المغالطات وكيفية التفريق بين المدارس المنطقية المختلفة.
